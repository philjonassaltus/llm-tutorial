Set up virtual env
`virtualenv --python=python3.12 ./venv`
Run virtual env
`python3 -m venv ./venv && source ./venv/bin/activate`
To install tensorflow
`python3 -m pip install tensorflow nltk numpy pandas matplotlib`
Finally
`python3 main.py`
